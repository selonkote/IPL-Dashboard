# IPL-Dashboard

1. Project Title
IPL Data Analysis Dashboard | Power BI**

2.  Short Description / Purpose
An interactive **Power BI** dashboard built to analyze Indian Premier League (IPL) match data and uncover valuable insights into team performance, player statistics, season trends, match outcomes, and venue analysis. The project demonstrates end-to-end data analysis, including data cleaning, modeling, DAX calculations, and interactive dashboard design for data-driven decision-making.

3.  Tech Stack
* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV
* Data Modeling

4. Data Source
Kaggle – Indian Premier League (IPL) Dataset

5. Highlights
* Interactive dashboard with dynamic filters and slicers
* Team & Player Performance Analysis
* Season-wise Trends and Match Insights
* Orange Cap & Purple Cap Statistics
* Toss Impact and Venue Analysis
* Business Intelligence dashboard with actionable insights
